# Student Management REST API

This project is built using Node.js HTTP module.

## Features
- CRUD operations (GET, POST, PUT, DELETE)
- Validation (email, required fields, year range)
- Error handling
- Bonus: createdAt and updatedAt timestamps

## API Endpoints
- POST /students
- GET /students
- GET /students/:id
- PUT /students/:id
- DELETE /students/:id
